## 🔌 硬件选型

### 主控芯片: ATK-ESP-01

**ESP8266 核心参数**:
| 参数 | 规格 |
|------|------|
| CPU | 80MHz (可超频到 160MHz) |
| Flash | 1MB |
| RAM | 50KB (可用) |
| Wi-Fi | 802.11 b/g/n |
| 工作电压 | 3.3V |
| 工作电流 | 80mA (正常) / 300mA (峰值) |
| GPIO | GPIO0, GPIO2 |
| 串口 | 1 路 UART |

**ESP-01 引脚定义**:
```
       ┌────────┐
  GND  │1     8│  VCC (3.3V)
GPIO2  │2     7│  RST
GPIO0  │3     6│  CH_PD (使能)
  RXD  │4     5│  TXD
       └────────┘
```

### 执行器: SG90 微型舵机

**SG90 参数**:
| 参数 | 规格 |
|------|------|
| 工作电压 | 4.8V - 6V |
| 工作电流 | 100mA (空载) / 500mA (堵转) |
| 扭矩 | 1.8kg·cm @ 4.8V |
| 速度 | 0.12sec/60° @ 4.8V |
| 控制信号 | PWM (50Hz) |
| 脉宽范围 | 0.5ms - 2.5ms |
| 转动角度 | 0° - 180° |
| 重量 | 9g |
| 尺寸 | 22.2 x 11.8 x 31mm |

**SG90 接线**:
```
红线 (VCC)  - 4.8V ~ 6V
棕线 (GND)  - 地
橙线 (Signal) - PWM 信号 (3.3V/5V 兼容)
```

**PWM 控制参数**:
```
角度对应脉宽:
  0°   = 0.5ms  (500μs)
  90°  = 1.5ms  (1500μs)
  180° = 2.5ms  (2500μs)

频率: 50Hz (周期 20ms)
```

## ⚡ 供电方案设计

### 方案一：USB 充电器供电（推荐）

```
USB 充电器 (5V/1A)
    │
    ├─→ SG90 VCC (5V)
    │       └─→ SG90 可承受 4.8V-6V
    │
    └─→ AMS1117-3.3 稳压模块
            │
            └─→ ESP-01 VCC (3.3V)

所有 GND 共地
```

**优点**:
- ✅ 成本低（USB 充电器通用）
- ✅ 电流充足（1A 足够）
- ✅ 安全可靠

**所需器件**:
- USB 充电器 (5V/1A)
- AMS1117-3.3 稳压模块
- 杜邦线

### 方案二：电池供电（便携版）

```
3.7V 锂电池 (2000mAh)
    │
    ├─→ 升压模块 (3.7V → 5V)
    │       │
    │       └─→ SG90 VCC (5V)
    │
    └─→ AMS1117-3.3 稳压模块
            │
            └─→ ESP-01 VCC (3.3V)
```

**优点**:
- ✅ 便携，不受电源线限制
- ✅ 可安装在任意位置

**缺点**:
- ❌ 需要定期充电
- ❌ 成本稍高

**所需器件**:
- 3.7V 18650 锂电池
- TP4056 充电模块
- MT3608 升压模块 (3.7V → 5V)
- AMS1117-3.3 稳压模块

## 🔗 电路连接详解

### 完整接线图

```
┌──────────────┐
│ 5V 电源       │
│              │
│  +5V   GND   │
└───┬─────┬────┘
    │     │
    │     ├──────────────┐
    │     │              │
    │  ┌──┴───┐      ┌───┴────┐
    │  │ SG90 │      │ ESP-01 │
    │  │      │      │        │
    │  │ VCC──┘      │        │
    │  │ GND─────────┤ GND    │
    │  │ Sig ────────┤ GPIO2  │
    │  └──────┘      │        │
    │                │ VCC ───┼─← 3.3V
    │                └────────┘
    │
    └──→ AMS1117-3.3 输入
             │
             └──→ 3.3V 输出
```

### GPIO 配置

| ESP-01 引脚 | 功能 | 连接 | 备注 |
|------------|------|------|------|
| **GPIO2** | PWM 输出 | SG90 Signal | 舵机控制 |
| **GPIO0** | 预留 | LED (可选) | 状态指示 |
| **RXD** | 串口接收 | USB-TTL RX | 烧录/调试 |
| **TXD** | 串口发送 | USB-TTL TX | 烧录/调试 |
| **RST** | 复位 | 按键 | 复位开关 |
| **CH_PD** | 使能 | VCC (3.3V) | 必须拉高 |

### ⚠️ 特别注意事项

#### 1. GPIO0 启动模式

ESP8266 启动时会检查 GPIO0 的电平：
- **GPIO0 = 高电平** → 正常运行模式
- **GPIO0 = 低电平** → 烧录模式

**建议配置**:
```
GPIO0 ─┬─→ 10kΩ 上拉电阻 ─→ VCC (3.3V)
       │
       └─→ 烧录按键 ─→ GND
```

#### 2. 供电稳定性

ESP-01 Wi-Fi 工作时会有瞬间大电流（300mA）：

**解决方案**:
```
3.3V ─→ 100μF 电容 ─→ ESP-01 VCC
              │
            ─┴─ GND
```

#### 3. 共地问题

**必须确保所有模块共地**:
```
5V电源 GND ──┬─→ SG90 GND
             ├─→ ESP-01 GND
             └─→ 稳压模块 GND
```

## 🔋 功耗分析

### 理论功耗

| 状态 | ESP-01 | SG90 | 总计 |
|------|--------|------|------|
| **待机** | 80mA @ 3.3V | 10mA @ 5V | ~0.3W |
| **Wi-Fi 传输** | 170mA @ 3.3V | 10mA @ 5V | ~0.6W |
| **舵机动作** | 80mA @ 3.3V | 500mA @ 5V | ~2.8W |

### 电池续航估算（便携版）

**使用 2000mAh 锂电池**:

```
待机时间:
  平均电流: 100mA
  续航: 2000mAh / 100mA = 20 小时

如果每小时动作 10 次，每次 1 秒:
  待机: 59分50秒 @ 100mA = 99.7mAh
  动作: 10秒 @ 600mA = 1.67mAh
  每小时总计: 101.37mAh

  续航: 2000mAh / 101.37mAh = 约 19.7 小时
```

## 🌡️ 散热设计

### 发热源

1. **AMS1117-3.3**:
   - 压差: 5V - 3.3V = 1.7V
   - 电流: 最大 170mA
   - 功耗: 1.7V × 0.17A = 0.29W
   - **需要散热片**

2. **ESP-01**:
   - 发热量较小
   - 自然散热即可

3. **SG90**:
   - 堵转时发热
   - 避免长时间堵转

### 散热方案

```
AMS1117-3.3 ─→ 贴散热片 (10mm x 10mm x 3mm)
外壳 ─→ 开散热孔 (直径 5mm x 4 个)
```

## 🧪 测试方案

### 阶段一：模块独立测试

1. **稳压模块测试**
   ```
   输入: 5V
   输出: 3.3V ± 0.1V
   负载: 200mA
   ```

2. **舵机测试**
   ```
   供电: 5V
   PWM: 50Hz, 0.5ms~2.5ms
   动作: 0° ↔ 180° 顺畅
   ```

3. **ESP-01 测试**
   ```
   供电: 3.3V
   串口: 115200 baud
   AT 指令响应正常
   ```

### 阶段二：集成测试

1. **供电稳定性**
   - 测量各模块电压
   - 观察 Wi-Fi 连接时电压波动

2. **舵机控制**
   - ESP-01 控制舵机转动
   - 检查 PWM 信号

3. **长时间运行**
   - 连续运行 24 小时
   - 监测温度和稳定性
