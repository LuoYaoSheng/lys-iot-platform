## [2.0.0] - 2025-12-19

### 🎯 重大改版 - 面向创客用户

**设计理念转变**:
- ❌ 移除所有技术细节(代码、Git、Docker 等)
- ✅ 面向普通创客,零技术门槛
- ✅ 聚焦产品价值而非技术实现

### 完全重写 ✨

**新增内容**:
1. **准备材料区** - 展示 4 样必需品及价格
   - ESP32 开发板(15-30元)
   - SG90 舵机(5-15元)
   - 连接线(5元)
   - Android 手机(免费 APP)
   - 购买渠道指引

2. **简化使用步骤** - 从 5 步减为 4 步
   - 连接硬件(强调不用焊接)
   - 烧写固件(5分钟搞定)
   - 手机配网(APP 自动找设备)
   - 远程控制(随时随地)

3. **视频教程区** - 3 个教程入口
   - 硬件连接教程
   - 固件烧录教程
   - APP 使用教程

4. **联系/帮助区** - 新增
   - 提交反馈入口
   - 常见问题解答

### 移除内容 ❌

- 移除"功能特性"区(太技术化)
- 移除"技术栈"区(用户不关心)
- 移除所有代码示例
- 移除 Git 仓库链接
- 移除 Docker Hub 链接
- 移除开发者相关内容

### 优化 🎨

- **语言风格** - 从技术文档改为产品介绍
- **用户视角** - 从"怎么开发"改为"怎么用"
- **成本透明** - 明确标注硬件价格(30-60元)
- **门槛降低** - 强调"无需编程、无需焊接"

### 定位 🎯

**目标用户**:
- 创客玩家
- 智能家居爱好者
- 学生
- DIY 爱好者

**核心价值**:
- 零技术门槛
- 超低成本(30-60元)
- 快速上手(5分钟)
- 立即可用
