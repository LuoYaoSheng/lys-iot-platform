## 📖 简介

这是 **IoT 智能设备管理平台** 的产品落地页,专为创客和普通用户设计,让他们能快速用 ESP32 和舵机搭建自己的智能控制系统。

### 核心理念

✅ **零技术门槛** - 无需编程、无需焊接
✅ **超低成本** - 30-60 元即可上手
✅ **快速上手** - 跟着视频 5 分钟搞定
✅ **立即可用** - 下载固件和 APP 就能玩

## 🎯 目标用户

- 🎨 **创客玩家** - 想玩智能硬件但不想写代码
- 🏠 **智能家居爱好者** - 想自己动手改造家居
- 🎓 **学生** - 学习物联网但觉得门槛太高

## 📋 页面结构

### 1. **首屏 Hero**
- 吸引眼球的标题:"让你的 ESP32 和舵机变智能"
- 突出特点:手机远程控制、5 分钟上手
- 强调门槛:无需编程、无需焊接
- CTA 按钮:下载 APP、观看教程

### 2. **准备材料**
- 展示 4 样必需品:ESP32、舵机、连接线、手机
- 标注价格:让用户知道成本很低(30-60元)
- 购买渠道:淘宝、京东等

### 3. **使用步骤**
简单 4 步流程:
1. 连接硬件(不用焊接)
2. 烧写固件(5分钟搞定)
3. 手机配网(APP 自动找设备)
4. 远程控制(随时随地操作)

### 4. **云端服务部署**
- 选项 1: 使用公共服务器(推荐新手,已默认配置)
- 选项 2: 自己部署服务器(进阶玩家,需联系作者获取服务端代码)
- 详细的 Docker 部署教程(5 个步骤)

### 5. **下载专区**
- Android APP 下载(链接到本地 releases 目录)
- ESP32 固件下载(链接到本地 releases 目录)

### 6. **视频教程**
- 硬件连接教程
- 固件烧录教程
- APP 使用教程
- 链接到 B站(待填入真实链接)

### 7. **联系我们**
- 常见问题解答
- Issue 反馈链接
