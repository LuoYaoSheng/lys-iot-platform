# Open IoT Platform - 简洁设计系统
> 作者: 罗耀生
> 版本: 3.0.0
> 更新日期: 2025-01-11

## 设计理念

**"简洁、专业、高效"**

- **单色调**: 以蓝色为主，配合中性灰
- **极简**: 去除多余装饰，突出内容
- **清晰**: 层次分明，易于理解
- **工程感**: 专业、可靠、高效

---

```
White:     #FFFFFF  (主背景)
Gray-50:   #F5F5F7  (次背景)

Gray-200:  #D1D1D6  (边框)

Gray-700:  #3A3A3C  (主要文字)
Gray-900:  #000000  (强调文字)
```
