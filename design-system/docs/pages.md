## 1. 启动页 (SplashPage)

### 路由
`/splash`

### 功能
- 显示品牌 Logo 和名称
- 检查登录状态
- 自动跳转（2秒）

### 布局

```
┌────────────────────────────────────┐
│                                    │
│                                    │
│              [LOGO]                │
│                                    │
│         Open IoT Platform          │
│                                    │
│                                    │
│                                    │
│                                    │
│                                    │
└────────────────────────────────────┘
```

### 设计规范

| 元素 | 规范 |
|------|------|
| 背景 | `gray50` (#F5F5F7) |
| Logo | 80x80pt，居中 |
| 品牌名 | `fontSizeTitle` (20pt)，`gray700` |

### 设计规范

| 元素 | 规范 |
|------|------|

| 第二行 | 产品信息、固件版本、位置 | 12pt / `gray500` |

```
┌────────────────────────────────────┐
│  ← 配置中                          │
├────────────────────────────────────┤
│                                    │
│         正在配置设备...             │
│                                    │

└────────────────────────────────────┘
```

### 设计规范

| 元素 | 规范 |
|------|------|
| 步骤指示器 | 4 步，每步 60pt 宽 |
| 完成状态 | ✓ 绿色 `success` |
| 进行中 | • 蓝色 `primary` |
| 待完成 | ○ 灰色 `gray300` |
| 进度文本 | `fontSizeBody` (16pt)，`gray500` |

### 配网状态

| 状态 | 进度 | 文本 |
|------|------|------|
| connecting | 25% | 正在连接设备... |
| sending | 50% | 正在发送配置... |
| waitingWifi | 75% | 等待设备连接 WiFi... |
| activating | 90% | 正在激活设备... |
| activated | 100% | 配置成功！ |
| error | - | 配置失败，请重试 |
