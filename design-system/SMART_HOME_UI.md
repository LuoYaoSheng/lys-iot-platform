# Open IoT Platform - 智能家居 UI 设计系统
> 作者: 罗耀生
> 版本: 2.0.0
> 更新日期: 2025-01-11

## 设计理念

**"让家更智能，让控制更简单"**

- **场景化**: 按房间/场景组织，符合用户生活习惯
- **直观化**: 一目了然的状态，简单的控制操作
- **舒适感**: 温暖的色调，柔和的动画，家的感觉
- **科技感**: 现代设计语言，可靠的品质感

---

## 1. 视觉风格

### 1.1 整体风格
- **渐变背景**: 营造空间感和舒适氛围
- **玻璃拟态**: 现代感与层次感
- **大圆角**: 柔和友好，减少生硬感
- **微动效**: 提升交互体验，不喧宾夺主

### 1.2 设计语言参考
- Apple HomeKit 的简洁
- 米家的场景化
- Yeelight 的温暖

### 3.1 设备卡片

#### 标准设备卡片 (大卡片)

```
┌─────────────────────────────────────────────┐
│  ┌─────┐                             🟢   │
│  │ 💡 │  客厅吸顶灯                       │
│  └─────┘  16°C · 湿度 45%                │
│                                             │
│  ┌─────────────────────────────────────┐   │
│  │  ○──────────────○────────○  75%     │   │

│  ┌────┐  ┌────┐  ┌────┐  ┌────┐          │
```

- **阴影**: `0 8px 30px rgba(0,0,0,0.08)`

## 6. 动画规范

### 6.1 过渡动画

| 场景 | 时长 | 缓动 |
|------|------|------|
| 页面切换 | 300ms | ease-in-out |
| 卡片展开 | 250ms | ease-out |
| 开关切换 | 150ms | ease-in-out |
| 加载动画 | 循环 | linear |

### 6.2 反馈动画

- **点击**: 缩放 0.95 持续 100ms
- **开关**: 颜色渐变 200ms
- **加载**: 呼吸效果
